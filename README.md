# odin_recipes
This project is part of my journey through The Odin Project.
Even though I’ve already learned HTML, CSS, JavaScript, and even React before, I decided to start again from the basics. Not because I have to—but because I want to rebuild my foundation and actually see and use what I know instead of just moving on too fast.
I also want this repo to feel real—not perfect, not over-polished, and not like something generated. Just honest progress.
What I Learned & Used in This Project
Linux Terminal: I’ve been using the linux terminal for this project. Navigating directories, managing files, and getting comfortable with basic commands.
Git Commands: I used Git to track changes, commit my work, and understand how version control can be a lifesaver. Commands like git clone, git add, git commit, and git push have become second nature.
GitHub Repository: I pushed my project to GitHub to store and share it. Using GitHub is an important step in making repos accessible, keeping it organized, and sharing it with others, which is a key part of any developer’s workflow.
HTML: I worked with basic HTML to structure the content of the recipe page. Even though HTML is one of the simplest languages, it’s powerful, and I’ve learned to appreciate its simplicity and flexibility in building web pages.
